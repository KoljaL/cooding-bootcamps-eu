Web App Basic

This is a basic web app with a menu and a toggle theme button.

The menu is a simple list of links. The toggle theme button is a label with a title and a background image.

The menu is hidden by default. It is shown when the menu button is clicked. The menu button is a label with a title and a background image.

The toggle theme button toggles between light and dark theme. The theme is stored in the local storage. The theme is set on page load.
